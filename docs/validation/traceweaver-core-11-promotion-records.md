# TraceWeaver Core 11 Promotion Records

This is the canonical source-to-target promotion inventory for the
TraceWeaver Core 11-skill suite.

Promotion records are required before any scrubbed public-candidate material is
promoted into public docs, `docs/distilled/`, public skill folders, schemas,
examples, runtime files, package manifests, or release claims.

Direct private-to-public promotion is prohibited for U4 public skill-folder,
U6 runtime/package, and U7 public-release acceptance. Exception review records
may document rejected or deferred alternatives, but they do not satisfy Core 11
public/live acceptance gates.
