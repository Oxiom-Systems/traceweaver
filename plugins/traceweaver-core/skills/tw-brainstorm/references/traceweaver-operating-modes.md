# TraceWeaver Operating Modes For tw-brainstorm

<!-- TRACEWEAVER: file-role=operating-mode-policy; req=REQ-TW-057; trace=TRACE-TW-037; ver=VER-TW-047 -->

`tw-brainstorm` operates in source-evidence mode. It may gather context,
challenge assumptions, and produce candidate needs or requirement text, but it
must route anything that could become authority through `tw-requirements-review`
before implementation planning or work.

Runtime, publication, remote mutation, clean replacement, and release claims
remain held.
