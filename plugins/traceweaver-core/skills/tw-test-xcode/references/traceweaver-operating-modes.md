# TraceWeaver Operating Modes For tw-test-xcode

<!-- TRACEWEAVER: file-role=operating-mode-policy; req=REQ-TW-057; trace=TRACE-TW-037; ver=VER-TW-047 -->

`tw-test-xcode` operates in verification-evidence mode only when the scheme,
target, requirement ID, trace ID, verification ID, expected signal, and artifact
location are clear. Otherwise Xcode output is exploratory or held evidence.
