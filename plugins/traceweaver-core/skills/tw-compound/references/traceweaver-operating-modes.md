# TraceWeaver Operating Modes For tw-compound

`tw-compound` operates in source-evidence and learning-capture mode. It may
create or update learning material through the packaged `ce-compound` engine,
but it must not promote that material into requirements, trace authority,
publication evidence, or release readiness without a separate reviewed gate.
